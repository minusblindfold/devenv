# Ghostty Cheatsheet

## Splits

| Key | Action |
|-----|--------|
| `Cmd+Shift+Right` | New split right |
| `Cmd+Shift+Left` | New split left |
| `Cmd+Shift+Down` | New split down |
| `Cmd+Shift+Up` | New split up |
| `Cmd+Right` | Navigate split right |
| `Cmd+Left` | Navigate split left |
| `Cmd+Down` | Navigate split down |
| `Cmd+Up` | Navigate split up |
| `Cmd+X` | Close pane |

## Other

| Key | Action |
|-----|--------|
| `Cmd+P` | Project picker (ctrl+p) |

## Settings

| Setting | Value |
|---------|-------|
| Theme | Dark Pastel |
| Font | JetBrainsMono Nerd Font 14pt |
| Bell | Disabled |
| Inactive split opacity | 0.4 |
| Split divider | Purple (#6a0dad) |
